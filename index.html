Instructions/Rules

- Each Player takes turns answering trivia Questions. Player One starts.


To Win;

- The first person to get 5 points (after taking the same number of turns) wins.

- If both players get 5 points after the same number of turns, the game goes to sudden death. The first person to miss a question (after the same number of turns) loses.

- If after all the questions available are answered, both players still have the same points, the game ends in a tie.
